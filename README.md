# auth-grails-vue

 #### To run the app, please clone repository, open folder with a project in terminal and run a command: 
 > ./gradlew bootRun -parallel
 #### Then open in browser link: 
 > localhost:3000
 #### On a login page you can use this data: 
 ```
 username: admin
 password: 1234
 ```
 #### or
 ```
 username: user
 password: 5678
 ```
